Cookie Clicker

klik op het koekje om een zo hoog mogelijke score te krijgen

Game Controls

klik op het koekje met linker muisklik om je score hoger te maken

Linker Muisklik ( op koekje ) = score omhoog
Linker Muisklik ( op reset ) = full reset
Linker Muisklik ( op power ups ) = power ups kopen

Parkour game

Help het rode rechthoek met parkour om zijn droom te halen van de finish vlag te halen!

Game Controls

Gebruik W, D en Spatiebar om rond te lopen en springen!

A = Beweeg naar links
D = Beweeg naar rechts
Spatiebalk = Spring de lucht in

Obstacle Maze

Verzamel 3 sleutels die verdeeld zijn over het volledige speelveld terwijl je obstakels ontloopt. Zodra je alle drie de sleutels hebt opgeraapt kan je naar de deur navigeren en uit het doolhof ontsnappen.

Game Controls

Beweeg je blokje doormiddel van WASD, verzamel sleutels en open deuren met E

W = Boven
A = Links
S = Beneden
D = Rechts
E = Sleutel oprapen of Deur openen

Snake

Dit is een basis versie van de game Snake. Je speelt als een groene slang die appels eet om langer te worden.

Game Controls

Beweeg door het vierkant om appels te eten. Het spel stopt als je de buitenkant of de slang zelf aanraak.

W = Boven
A = Links
S = Beneden
D = Rechts
R = Herstart Spel
