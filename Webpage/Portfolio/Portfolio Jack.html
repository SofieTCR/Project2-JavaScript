Portfolio Jack van Bommel

Over Mij

Mijn naam is Jack van Bommel. Ik ben 18 jaar oud en ik woon in Rottedam. Ik houd veel van computers en technologie, daarom heb ik gekozen voor TCR en de Software Development opleiding. Ik hoop later aan de slag te kunnen gaan als Game Developer.

Mijn Vaardigheden

Mijn sterkste kant op dit moment is HTML en CSS. Ik kan simpele websites maken in een redelijk snelle tijd zonder al te veel bugs. Ik ben op dit moment bezig met het leren van Javascript en PHP. Ik hoop mijn programmeer kennis uiteindelijk zover te ontwikkelen dat ik nieuwe talen snel kan oppakken zonder al te veel moeite.

Buiten programmeren om heb ik nog meer vaardigheden. Ik ben erg sociaal en kan goed samenwerken. Ik ben erg leergierig en zal kansen om iets nieuws te leren nooit laten liggen. Verder ben ik in staat simpele ICT problemen op te lossen en kan ik goed google gebruiken.

Mijn Ervaring

Ik heb tot nu twee volledig werkende websites gemaakt met HTML en CSS. Ik heb een beetje ervaring met Python en simpele programma's hiermee gemaakt in het verleden. Tot slot heb ik voor deze website een simpele parkour game gemaakt met Javascript. Ik hoop meer projecten te maken in de toekomst, op school, stage en daar buiten.
