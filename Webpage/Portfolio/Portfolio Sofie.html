Portfolio Sofie Brink

Over Mij

Mijn naam is Sofie, Ik ben 18 jaar en ik volg momenteel de opleiding software developer op het Techniek College Rotterdam. Ik heb gekozen voor het TCR omdat het een van de weinige scholen is die een software developer opleiding bied en in vergelijking met de andere school, het TCR mijn gezelliger leek.

Mijn Vaardigheden

De taal die ik op dit moment het beste beheers is C# maar naast deze taal heb ik ook verstand van Javascript, HTML/CSS, PHP C++. Dus dankzij deze kennis kan ik met relatief gemak een website aanmaken die goed functioneerd. Ook helpen talen zoals C++ bij het programmeren van microcontrollers zoals een arduino of ESP32. En een taal zoals C# kan ik gebruiken om addons voor veel moderne games te maken, iets wat ik erg leuk vind om te doen in mijn vrije tijd.

Mijn Ervaring

Tot nu toe heb ik 4 websites en 8 addons gemaakt met de kennis die ik heb opgedaan op en buiten school. Ik vind voornamelijk de back end van webdev en game development het leukst dus ik hoop in de toekomst hier meer mee te kunnen doen binnen school, in mijn vrije tijd en op stage.
